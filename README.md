# WRTNet-EFWI
Official PyTorch implementation of WRTNet-EFWI for physics-informed unsupervised elastic full-waveform inversion.

This repository accompanies the manuscript:

“A Wavelet-Refined Transformer Network for Physics-Informed 
Unsupervised Elastic Full-Waveform Inversion.”

## Current Release

The current review-stage release provides the baseline Transformer 
implementation and the essential environment information for 
physics-informed elastic full-waveform inversion.

The complete WRTNet-EFWI implementation, including the wavelet 
refinement module, training configurations, and reproduction scripts, 
will be released upon acceptance of the manuscript.

## Code Availability

The source code is provided for academic research and reproducibility. 
The complete implementation and trained model checkpoints will be 
released after manuscript acceptance.
